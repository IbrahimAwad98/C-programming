
## Om detta repository

Detta repository innehåller en samling C-programmeringsprojekt som utvecklats under min utbildning och personliga inlärning. Projektet inkluderar olika typer av applikationer från enkla övningar till mer komplexa projekt och spel.

## Struktur

Repositoryt är organiserat i följande huvudsakliga mappar:

### Client App
Innehåller klientapplikationer utvecklade i C.

### Games
En samling spel implementerade i C-programmering, demonstrerande olika programmeringskoncept och spellogik.

### Laborationer
Laborationsuppgifter och övningar från kursen. Här finns praktiska uppgifter som täcker olika aspekter av C-programmering.

### Project 1
Det första större projektet som implementerar mer avancerade C-programmeringstekniker.

### Project 2
Det andra större projektet med ytterligare komplexitet och funktionalitet.

## Teknologi

- **Programmeringsspråk:** C
- **Utvecklingsperiod:** Januari 2025 - Pågående

## Komma igång

### Förutsättningar

För att kompilera och köra programmen i detta repository behöver du:

- GCC (GNU Compiler Collection) eller annan C-kompilator
- Make (rekommenderas för vissa projekt)
- Unix-baserat system (Linux/macOS) eller Windows med lämplig utvecklingsmiljö

### Kompilering

Generell kompilering av C-filer kan göras med:

```bash
gcc -o programnamn filnamn.c
./programnamn
```

Specifika kompileringsinstruktioner kan finnas i respektive projektmapp.
